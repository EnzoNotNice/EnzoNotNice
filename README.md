## 🚀 About Me

I design and build Discord bots — from lightweight utility bots to full community platforms — using **JavaScript/TypeScript** and **Node.js**, with **Python** for scripting and automation. I care about clean architecture, readable code, and features that still hold up once real users (and real edge cases) show up.
